Папка с файлами, необходимыми для установки драйверов Lcard и проверки корректности их подключения.
1. Lcomp : папка с драйверами для Lcard.
2. Lusbapi34_dec2023 : cpp библиотеки для общения с Lcard.
3. ModuleViewer : Программа для проверки наличия подключения к Lcard.
4. LGraph2 : Программа для отображения данных с Lcard.
5. Firmware E20-10 : Папка с прошивками для Lcard.

Исполняемым файлом (файлом, с которого происходит запуск) является KWR102/filament_and_anode - copy.py

При нажатии кнопки start на графическом интерфейсе происходит запуск исполняемой таблицы, описанной в CommandTable_example.ini.

Конкретно в примере CommandTable_example.ini, таблица будет работать 12 секунд в течение этого времени отдавая команды на Корад.

После завершения команд таблицы Корад и Lcard останавливаются, ток и напряжение Корада ставится на ноль.

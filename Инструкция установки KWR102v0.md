Инструкция по запуску программы для Korad и Lcard на новом оборудовании.

Требования: ОС поддерживает python версии > 3.8.

Инструкция:

1. Скачать и установить python версии > 3.8. 
Например, можно отсюда https://www.python.org/downloads/ 

2. Установить библиотеки для python. 
Для приложения необходимы: pandas, numpy, matplotlib, PyQt5. 
Например, можно так:
2.1 Открыть командную строку, исполнить команды:
2.2 pip install pandas
2.3 pip install numpy
2.4 pip install matplotlib
2.5 pip install pyqt5
2.6 pip install pyserial

3. Скачать проект с github (https://github.com/ISA-f/AutoomaticLab/tree/main), рабочие версии собраны в отдельные папки Assemble_vN.

4. Загрузка драйверов для Lcard:
4.1 Перейти в папку Instrument Drivers and Installation\Installation Lcard\. 
4.2 Запустить Installation Lcard\Lcomp\lcomp_2023.exe.
4.3 Через программу, устанавливаемую из \Installation Lcard\LGraph2\Lgraph2_v2.36.2\setup.exe, можно проверить, что общение с Lcard корректно.

5. Загрузка драйверов для Korad:
5.1 Korad общается с компьютером через эмулируемый COM-порт, к которому мы подключаемся с помощью pyserial. Скачать pyserial например можно так:
5.1.1 Открываем командную строку.
5.1.2 pip install pyserial
5.2 Соответствующия файлы для проверки подключения к Korad лежат в папке Instrument Drivers and Installation\Installation KWR102 V3.4.

6. Проект готов к запуску. Папки Assemble_vN, соответствующие стабильным версиям программы, снабжены собственным описанием, далее необходимо выбрать версию, найти и запустить исполняемый файл,содержащийся в описании выбранной версии.
